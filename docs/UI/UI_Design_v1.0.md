# 慕思物模型平台 - UI/UX设计规范 v2.0

> 基于PRD v1.0更新，支持产品状态管理、物模型动态编辑

## 1. 设计概述

### 1.1 设计目标
为慕思物模型管理平台提供专业、易用的管理后台界面

### 1.2 设计风格
- **风格**: Minimalism & Enterprise Dashboard（简洁企业级仪表盘）
- **适用场景**: 企业级IoT设备管理平台
- **主色调**: #3B82F6 (蓝色)
- **背景色**: #F8FAFC (浅灰白)

## 4. 交互规范

### 4.1 过渡动画

| 类型 | 时长 | 缓动函数 | 场景 |
|-----|------|---------|------|
| 快速反馈 | 150ms | ease-out | 按钮悬停、焦点 |
| 标准过渡 | 200ms | ease-in-out | 卡片悬停、展开 |
| 页面过渡 | 300ms | ease-in-out | 弹窗、抽屉 |

### 4.2 状态反馈

```
加载中: 骨架屏动画
成功:   绿色Toast提示 + 图标
失败:   红色Toast提示 + 错误信息
确认:   二次确认弹窗
```

### 4.3 产品状态交互

| 操作 | 开发中状态 | 已发布状态 |
|-----|----------|----------|
