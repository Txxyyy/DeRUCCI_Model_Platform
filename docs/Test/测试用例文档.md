## 1. 测试概述

### 1.1 测试策略

| 测试类型 | 覆盖率目标 | 测试方法 | 工具 |
|---------|-----------|---------|------|
| 单元测试 | > 70% | 白盒测试 | JUnit/Mockito |
| 集成测试 | > 50% | 黑盒测试 | Spring Test |
| 接口测试 | 100% | API测试 | Postman/JMeter |
| UI测试 | 核心流程 | 自动化 | Selenium/Cypress |
| 性能测试 | 关键接口 | 负载测试 | JMeter |
| 安全测试 | 渗透测试 | 黑盒/白盒 | Burp/AWVS |

|-----|------|--------|------|
