## 8. 自定义参数扩展

所有模板均支持以下扩展方式：

### 8.1 添加自定义属性

```json
{
  "identifier": "custom_pressure_sensor",
  "name": "自定义压力传感器",
  "dataType": "int",
  "valueRange": { "min": 0, "max": 1000 },
  "unit": "Pa",
  "accessMode": "readOnly",
  "description": "用户自定义添加的压力传感器"
}
```

### 8.2 添加自定义事件

```json
{
  "identifier": "custom_alarm",
  "name": "自定义告警",
  "eventType": "alarm",
  "outputData": [
    { "identifier": "code", "name": "告警码", "dataType": "int" },
    { "identifier": "message", "name": "告警信息", "dataType": "string" }
  ]
}
```

### 8.3 添加自定义命令

```json
{
  "identifier": "custom_control",
  "name": "自定义控制",
```

## 9. 模板使用指南

### 9.1 基于模板创建产品

1. 在物模型管理页面选择"基于模板创建"
2. 选择适合的模板（如"智能床垫基础模板"）
3. 根据实际需求调整属性、事件、命令
4. 保存并发布物模型

### 9.2 模板版本管理

- 每个模板有独立的版本号
- 模板更新会创建新版本
- 已关联产品的模板不支持删除

### 9.3 模板定制建议

| 场景 | 建议 |
|-----|------|
| 新产品与模板完全匹配 | 直接使用模板，无需修改 |
| 产品有少量差异 | 基于模板创建，自定义少量属性 |
| 产品有较大差异 | 基于模板创建，或从零创建 |
| 通用设备 | 使用标准模板（温度传感器、开关等） |
